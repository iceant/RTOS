# 优先级划分

## 三位可用优先级

当 `__NVIC_PRIO_BITS =3`

| BIT7 | BIT6 | BIT5 | BIT4 | BIT3 | BIT2 | BIT1 | BIT0 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
|      |      |      | --   | --   | --   | --   | --   |

**可用值**

| 二进制   | 十六进制 |
| -------- | -------- |
| 00000000 | 00       |
| 00100000 | 20       |
| 01000000 | 40       |
| 01100000 | 60       |
| 10000000 | 80       |
| 10100000 | A0       |
| 11000000 | C0       |
| 11100000 | E0       |

## 四位可用优先级

当 `__NVIC_PRIO_BITS =4`

| BIT7 | BIT6 | BIT5 | BIT4 | BIT3 | BIT2 | BIT1 | BIT0 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
|      |      |      |      | --   | --   | --   | --   |

**可用值**

| 二进制   | 十六进制 |
| -------- | -------- |
| 00000000 | 00       |
| 00010000 | 10       |
| 00100000 | 20       |
| 00110000 | 30       |
| 01000000 | 40       |
| 01010000 | 50       |
| 01100000 | 60       |
| 01110000 | 70       |
| 10000000 | 80       |
| 10010000 | 90       |
| 10100000 | A0       |
| 10110000 | B0       |
| 11000000 | C0       |
| 11010000 | D0       |
| 11100000 | E0       |
| 11110000 | F0       |
